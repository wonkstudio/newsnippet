## 5. 사용 규칙

**해도 되는 것:**
- SVG 파일을 어떤 사이즈로든 확대·축소해서 사용
- 흰 배경 위에 로고 배치
- 네이비(#1C2B4A) 배경 위에 로고 배치

**하면 안 되는 것:**
- 로고 색상 임의 변경
- 로고 비율 왜곡 (가로·세로 따로 늘리기)
- 로고 위에 다른 요소 겹치기
- 저해상도 이미지를 확대해서 사용

## 6. PNG 변환 방법

앱스토어 등록 시 PNG가 필요한 경우:

**방법 1 — 브라우저에서 변환**
1. SVG 파일을 브라우저로 열기
2. 우클릭 → 이미지를 다른 이름으로 저장

**방법 2 — 온라인 변환 도구**
- https://svgtopng.com
- https://cloudconvert.com/svg-to-png

**방법 3 — Claude에게 요청**
- "newsnippet_logo.svg를 PNG로 변환해줘" 라고 하면 돼요
