## 3. 기술 스택

### 전체 자동화 흐름
```
매일 오전 5시 (UTC 20:00)
GitHub Actions 자동 실행
→ 연합뉴스 RSS 수집 (주요·경제·정치 피드)
→ 헤드라인 모음·[표]·[속보] 자동 필터링
→ Gemini가 40-50대 중요 뉴스 5개 필터링
→ AI 요약 + 용어설명 + 원문링크 + 에디터 비평 생성
→ 기존 뉴스 news_archive로 이동
→ Supabase news 테이블에 새 뉴스 5개 저장
→ GitHub Pages (index.html) 자동 업데이트
```

### 기술 스택 상세

| 영역 | 기술 | 비용 |
|------|------|------|
| 뉴스 수집 | RSS 피드 (연합뉴스) | 0원 |
| AI 필터링·요약 | Gemini API (gemini-2.5-flash) 유료 | 하루 몇 원 |
| 데이터베이스 | Supabase | 0원 |
| 자동화 | GitHub Actions (매일 오전 5시) | 0원 |
| 웹 배포 | GitHub Pages | 0원 |
| 앱 프레임워크 | React Native + Expo | 0원 |
| 앱 빌드 | EAS Build | 0원 |

### GitHub 저장소
```
wonkstudio/newsnippet (public)
├── .github/workflows/main.yml
├── index_final.js
├── index.html
└── privacy.html
```

**주소:**
