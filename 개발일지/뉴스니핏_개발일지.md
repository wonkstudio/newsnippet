## 1. 사업 기획

### 1.1 시장 기회
- 국내 40–50대 스마트폰 사용자 약 1,800만명
- 기존 뉴스 요약 서비스(뉴닉·어피티)는 2030 타겟 — **40–50대 특화 앱 부재**
- 구독자 1만명 달성 시 월 매출 약 3,900만원 (앱스토어 수수료 전)

### 1.2 타겟 페르소나

**페르소나 A — 직장인 김부장 (48세)**
- 출퇴근 시간 5분 안에 핵심 뉴스 파악 원함
- 경제·부동산·주식 정보에 민감
- 긴 기사 읽기 피로감 높음

**페르소나 B — 자영업 이사장 (52세)**
- 정치·사회 흐름이 사업 매출에 직결
- 신뢰할 수 있는 출처 중요시
- 복잡한 정책 이슈를 쉽게 이해하길 원함

**공통 페인 포인트**
- 네이버·다음 뉴스의 정보 과부하
- 낚시성 제목·편향 보도 불신
- 경제 용어(CPI, 기준금리 등) 설명 서비스 부재
- 무관한 알림으로 뉴스 앱 삭제 경험

### 1.3 2026 전략 보고서 기반 인사이트
> "디지털 전환의 명암: 기술 고도화와 인적 접근성의 균형을 통한 시장 선점 전략" 반영

## 2. 핵심 기능

### 2.1 MVP 기능 (현재 구현 완료)

| 기능 | 설명 |
|------|------|
| AI 3줄 요약 | 정치·경제 기사를 핵심 3줄로 압축 |
| 아침 브리핑 | 매일 오전 7시 뉴스 5개 카드 형태 제공 |
| 나에게 어떤 의미? | 단순 요약이 아닌 실생활 영향 해석 |
| 검증 배지 | 연합뉴스·조선일보 등 신뢰 언론사 표시 |
| 난이도 배지 | 쉬움·보통·심화 표시로 부담 감소 |
| 읽음 진행률 | 오늘 N/5 완료 진행 바 표시 |
| 용어 설명 | 경제 용어 탭하면 쉬운 설명 팝업 |
| 원문 링크 | 출처 언론사 원문 기사 바로가기 |
| 카카오톡 공유 | 요약 카드 텍스트 복사 + 뉴스니핏 브랜드 삽입 |
| 저장 기능 | 중요 뉴스 북마크 저장 |
| 글씨 크기 조절 | 작게·보통·크게 3단계 (40–50대 배려) |
| 알림 시간 설정 | 시·분 선택 모달로 원하는 시간 설정 |

### 2.2 로그인·접근 전략 (Freemium)

| 단계 | 제공 내용 |
|------|----------|
| 비로그인 맛보기 | 뉴스 카드 3개 + 상세 보기 1회 무료 |
| 두 번째 상세 시도 | 잠금 화면 + 카카오 가입 유도 |
| 카카오 로그인 (무료) | 뉴스 5개 전체 + 상세 무제한 |
| 프리미엄 (월 3,900원) | 무광고 + 저장 + 알림 설정 + 전체 기능 |

> **핵심 전략:** 카카오 로그인 한 번으로 가입 완료 — 절차적 복잡성(슬러지) 최소화

### 2.3 성장 기능 로드맵

| 시점 | 추가 기능 |
|------|----------|
| 4개월차 | TTS 음성 듣기 (Google TTS 무료) |
| 5개월차 | 키워드 맞춤 필터 |
| 6개월차 | 뉴스 아카이브 (주간·월간 정리) |
| 구독자 3,000명 | 카카오톡 채널 → 앱 내 커뮤니티 전환 |
| 구독자 1만명 | 전문가 코멘트, 주제별 게시판 |

## 3. UI/UX 디자인

### 3.1 디자인 원칙 (40–50대 최적화)
- **색상:** 네이비(#1C2B4A) + 하늘색(#EAF3FB) — 신뢰감 + 눈에 편안함
- **헤더:** 네이비 배경, 하단 둥글게 처리 — 무게감 있지만 부드러움
- **글씨:** 크고 굵게 — 폰트 크기 3단계 조절 가능
- **카드:** 여백 넉넉하게 — 한 눈에 파악
- **언어:** "~했습니다" 경어체 — 2030 앱과 차별화
- **"AI 요약" 표현 제거:** "핵심 정리"로 대체 — 40–50대 AI 거부감 해소

### 3.2 화면 구성

```
로그인 화면
├── 카카오톡으로 시작하기
└── 맛보기 (카드 3개·상세 1회)

홈 화면
├── 헤더 (앱 이름·날짜·진행률 바)
├── 뉴스 카드 목록
│   ├── 카테고리 배지 (경제/정치)
│   ├── 난이도 배지 (쉬움/보통/심화)
│   ├── 검증 배지 (★ 검증)
│   ├── 카드 번호 (1/5)
│   ├── 제목
│   ├── 3줄 요약
│   └── 자세히 보기 · 카톡 공유 버튼
└── [게스트] 잠금 배너 + 가입 유도

상세 화면
├── 헤더 (뒤로가기·공유·저장)
├── 핵심 정리 (1·2·3)
├── 나에게 어떤 의미?
├── 용어 설명 (탭하면 펼쳐짐)
├── 원문 기사 보기
└── 카카오톡으로 공유하기

저장 화면
└── 북마크한 뉴스 목록

설정 화면
├── 글씨 크기 (작게·보통·크게)
├── 관심 분야 선택
├── 알림 시간 설정 (시·분 모달)
└── 구독 플랜 정보
```

## 5. 카카오톡 채널 자동화 엔진

### 5.1 구조
```

→ RSS 피드에서 뉴스 수집 (연합·네이버·조선·한겨레)
→ Gemini API로 3줄 요약 생성
```

- **카카오톡 채널:** 무료 콘텐츠 선배포 → 앱 설치 유도. 40–50대 주 커뮤니케이션 채널
