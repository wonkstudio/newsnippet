# 🎨 Wonk Wiki

**Wonk Wiki**는 AI 이미지 생성 프롬프트, 스타일 가이드, 그리고 디자인 인사이트를 체계적으로 기록하고 관리하는 [Obsidian](https://obsidian.md/) 기반의 지식 보관함(Vault)입니다.

## 📂 주요 내용 (Contents)

- 📝 **AI 프롬프트 레퍼런스 (Prompt Reference)**
  - Midjourney, DALL-E 3, Stable Diffusion 등 다양한 생성형 AI 모델을 위한 정교한 프롬프트를 기록합니다.
  - 테마별, 아트 스타일별, 분위기별로 태그를 분류하여 직관적인 검색을 지원합니다.
- 🖼️ **구조 분석 및 인사이트 (Structure Analysis & Insight)**
  - 훌륭한 AI 아트의 핵심 구조(인물, 배경, 스타일, 텍스트 등)를 분해하고 분석합니다.
  - 실무에 바로 적용할 수 있는 디자인 팁과 재사용 가능한 프롬프트 스니펫을 정리합니다.
- 🔗 **상호 연결된 지식망 (Linked Knowledge)**
  - 옵시디언의 쌍방향 백링크(Backlink) 기능을 적극 활용하여 스타일 가이드, 컬러 팔레트, 타이포그래피 설계가 촘촘히 연결된 두 번째 뇌(Second Brain)를 구축합니다.

## 🚀 시작하기 (Getting Started)

1. [Obsidian](https://obsidian.md/download)을 다운로드하여 설치합니다.
2. Open folder as vault를 선택하고, 이 리포지토리를 클론(Clone)한 폴더를 지정합니다.
3. 노트들을 탐색하며 AI 아트워크의 인사이트를 발견하고 여러분만의 프롬프트를 확장해 보세요!
