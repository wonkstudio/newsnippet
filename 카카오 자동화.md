# 뉴스니핏 자동화 엔진

## 실행 순서

### 1단계 — Gemini API 키 발급 (무료)
1. https://makersuite.google.com 접속
2. "Get API Key" 클릭
3. 발급받은 키를 index.js의 GEMINI_API_KEY에 입력

### 2단계 — 카카오 채널 개설 & 토큰 발급
1. https://business.kakao.com 접속
2. "채널 만들기" → 뉴스니핏 채널 개설
3. https://developers.kakao.com 에서 앱 등록
4. 카카오 로그인 후 액세스 토큰 발급
5. KAKAO_ACCESS_TOKEN에 입력

### 3단계 — 실행
```bash
node index.js
```

### 4단계 — 서버에 올려서 자동화
- Vercel / Railway / Render 무료 플랜에 올리기
- scheduleDaily() 주석 해제하면 매일 자동 실행

## 구조
RSS 수집 → Gemini 요약 → 카카오 발송
